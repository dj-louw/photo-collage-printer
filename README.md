# Smarter At‑Home Photo Printing

## Lay out multiple photos on a single page to avoid wasted paper and ink.

A simple HTML+JS+CSS solution that allows you to arrange a collection of photos on a single page, to send to the printer. This reduces paper waste, and makes at-home printing of happy snaps a lot more convenient.

The goal here was to have a local and mobile friendly app that does not come with unnecesary bloat. There are many apps and software out there that can do this, but they all come with extra unnecesary stuff, and often they are cloud based which is a privacy concern. This app lives in the browser, there is no backend or cloud and it's only purpose is to allow the user to arrange photos on a page for printing.
