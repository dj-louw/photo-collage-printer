# Implementation Plan

This file documents the implementation steps for the Photo Collage Printer web-app, including folder structure, features, and deployment.

## Folder Structure
- `src/` - Application code (future expansion)
- `public/` - Static assets (HTML, CSS, JS)
- `docs/` - Documentation

## Features Implemented
- Project setup and folder structure
- Static web-app initialized (HTML, CSS, JS)
- Document management: add/remove pages, change page size
- Photo management: import, arrange, drag photos
- Responsive design (basic)
- Print/export via browser print utility

## Next Steps
- Add cropping, rotation, resizing, and removal controls for photos
- Improve responsive design for mobile/tablet
- Enhance export options (PDF, advanced print layout)
- Expand documentation

## Deployment
- Ready for static hosting (e.g., GitHub Pages)
- To deploy: copy contents of `public/` to hosting platform

For further refinement, update this plan as features are added or improved.
