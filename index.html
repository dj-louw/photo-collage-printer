How to use Photo Collage Printer
Add a new image to the active page.
Add a new blank page below the current one.
Delete the active page.
Delete the selected photo.
Rotate the selected photo 90° clockwise.
Toggle crop mode to adjust the visible part of a photo.
Drag resize handles to resize photos or zoom while cropping.
Open settings to switch between A4 and Letter.
Export all pages to a printable PDF.
